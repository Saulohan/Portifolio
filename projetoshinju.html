Shinju


Esse é meu primeiro projeto e ele é uma pokedex que pega as informações de cada pokemon em uma API e usando html,css e java script é feita toda formataçãoe alocação de informações e imagens dos pokemons ate a geração atual. Nas imagens abaico pode-se ter uma noção melhor de como o projeto funciona.pode se ver nas imagens abaixo.


Cada geração é dividida da outra.


Na imagem abaixo pode-se ver a primeira geração.


Card:


Cada card de shinju é apresentado da forma abaixo mudando e a cor para cada tipo que são: Primaria->Azul , Secundaria -> Verde e Lendária -> Laranja.


Modal:


Quando um card é prescionado abre um modal no qual mostra todas as informaçoes referentes a aquela shinju, informaçoes essas que podem ser: Habilidades, Poderes e Truques que o portador recebe ao utilizar a mesma.
